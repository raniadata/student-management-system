# Student Management System

## Description

This is a simple Student Management System built with Python.

The application allows users to manage student records using a command-line interface. All student data is stored in a JSON file, making the data persistent between program executions.

## Features

* Add a new student
* Show all students
* Search student by name
* Update student information
* Delete a student
* Input validation
* Store data using JSON

## Technologies Used

* Python 3
* JSON
* Git
* GitHub

## Project Structure


student-management-system/

## How to Run

1. Open the project in VS Code.
2. Run the `main.py` file.
3. Use the menu to manage students.
